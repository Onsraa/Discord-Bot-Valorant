# Discord-Bot-Valorant

As a developer, you can use as you want my discord bot to organize your Valorant custom games.

**0/ Features 

**************Commands**************
1 - /custom : Generate a message that organize 
2 - /random : Roll over the maps of Valorant 
************************************
